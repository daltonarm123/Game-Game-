# Game Design Notes (Draft)

## Core Principles
- Tick-based progression
- Fair, understandable combat math
- Fast UX on web and mobile
- Data-driven balancing with telemetry

## Combat Targets
- Deterministic base formulas
- Configurable seasonal modifiers
- Optional calibration from known-hit data

## Economy Targets
- Predictable upkeep and growth
- Prevent runaway snowballing
- Incentivize strategic specialization
