# Fallback Video Shotlist (2 Minutes)

Use this when live demo conditions are risky (bad wifi, unstable laptop environment).

## Recording Setup

1. Use OBS, Loom, or Xbox Game Bar.
2. Record at 1080p.
3. Keep mouse movement slow and intentional.
4. Use local fast demo mode (`LOCAL_DEMO_FAST=1`).

## Shot Order

1. **Intro (10 sec)**
- Show header + main layout.
- Say: "This is Crownforge, a modern tick-based kingdom strategy game."

2. **Overview Live Data (20 sec)**
- Open `Overview`.
- Show live rank, networth, resources, and queues.

3. **Train Troops (25 sec)**
- Open `Train Troops`.
- Queue 1 footman.
- Show it move from `train` toward `home`.

4. **Attack Kingdom (30 sec)**
- Open `Attack Kingdom`.
- Send 1 footman at target.
- Show result panel.

5. **War Room Evidence (25 sec)**
- Open `War Room`.
- Show `home/train/away` and `Recent Battles`.

6. **Close (10 sec)**
- Return to Overview.
- Say: "Core loop is live; funding accelerates full content and production systems."

## Deliverables

1. One full 2-minute clip.
2. One 45-second shortened clip for chat/social.
3. Optional subtitle burn-in for silent playback.
